# JSPM playground

## Getting started

This branch will follow some simple steps to get you up and running with jspm at a basic level.
The final structure of the project will look like:

```
jspm-playground/
|-- app/
    |-- client/
        |-- config.js
        |-- jspm_packages/
        |-- client.js
        |-- index.html
    |-- index.js
|-- node_modules/
|-- package.json
|-- README.md
```
